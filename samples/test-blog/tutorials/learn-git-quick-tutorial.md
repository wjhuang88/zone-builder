# Learn Git in 30 Minutes

Git is a distributed version control system that is widely used in software development. This tutorial will teach you the essential Git commands you need to know to get started.

## Basic Commands

### Initialize a Repository

```bash
git init
```

This command creates a new Git repository in the current directory.

### Stage and Commit Changes

```bash
git add .
git commit -m "Initial commit"
```

These commands stage all changes and commit them with a message.

### Check Status and History

```bash
git status
git log
```

These commands help you see the current status and commit history.
